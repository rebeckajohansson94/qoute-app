# 📱 Quote App

A simple React Native app built with Expo.

This project was created to practice core React Native concepts and working with APIs in a mobile environment.

---

## ✨ About the Project

The app fetches random quotes from a public API (https://dummyjson.com/docs/quotes) and displays them in a clean and simple interface.
